sequenceDiagram
    participant A as Application
    participant K as Kernel
    activate A
    loop wait until data is available
        A->>K: system call
        deactivate A
        activate K
        Note right of K: wait for data
        Note left of A: polling
        K-->>A: no data available
    end
    deactivate K
    activate A
    A->>K: system call
    deactivate A
    activate K
    Note right of K: copy data
    K-->>A: returned data
    deactivate K
    activate A
    deactivate A
